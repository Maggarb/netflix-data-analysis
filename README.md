# Netflix Data Analysis 🎬

## Objective
Analyze Netflix catalog trends using Python data analysis tools.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Netflix library heavily consists of movies
- Massive expansion after 2016
- USA dominates production
- Drama is the most frequent genre

## How to Run
pip install -r requirements.txt
jupyter notebook
